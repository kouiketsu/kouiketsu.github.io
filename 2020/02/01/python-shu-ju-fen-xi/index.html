numpy基础

什么是数据分析？

  • 数据分析是指用适当的统计分析方法对收集来的大量数据进行分析，将它们加以汇总和理解并消化，以求最大化地开发数据的功能，发挥数据的作用。

数据分析常用的库

  • 1、NumPy (Numerical Python) 基础数值算法
  • 2、SciPy(Scientific Python) 科学计算
  • 3、matplotlib 数据可视化
  • 4、pandas 序列高级函数

用于数组的文件输入和输出

  • numpy能够读写磁盘上的文本数据或二进制数据

    将数组以二进制格式保存到磁盘

    • np.load 读
    • np.save 写
    • np.savez 写多个

    获取文本文件

    • 可选选项: 指定各种分隔符、针对特定列的转换器函数、需要跳过的行数etc
    • np.read_csv 读取逗号分隔符文件
    • np.read_table
    • np.loadtxt 将数据加载到numpy数组中
    • np.genfromtxt 类似loadtxt
    • np.savetxt 与loadtxt相反,将数组写到文件中

线性代数

  • 线性代数(linear algebra)是关于向量空间和线性映射的一个数学分支。它包括对线、面和子空间的研究，同时也涉及到所有的向量空间的一般性质。
  • x.dot(y) == np.dot(x, y)(用于矩阵乘法) dot既是一个数组方法也是numpy函数

随机数生成

  • numpy.random模块
    • normal 标准正态分布的4*4样本数组
      • np.random.normal(size=(4, 4))

范例-随机漫步

  • 例子： 从0开始,步长1和-1出现的概率相等,实现1000步的随机漫步

    • 1、纯Python实现

      import random
position = 0
walk = [position]
steps = 1000
for i in range(steps):
  step = 1 if random.randint(0, 1) else -1
  position += step
  walk.append(position)

    • 2、numpy实现

  • 例子：首次穿越时间，即随机漫步过程中第一个到达某个特定值的时间

一次模拟多个随机漫步

  • pandas引入约定
    from pandas import Series, DataFrame
import pandas as pd

pandas的数据结构介绍

  • 两个主要数据结构

    • Series
    • DataFrame

  • Series

    • 类似于一维数组的对象
    • 由一组数据(各种numpy数据类型)和与之相关的数据标签(索引)组成
    • 字符串表现形式：索引在左,值在右
    • 不指定索引(自动创建一个0到len-1的整数型索引)
    • 可以通过索引取单个或多个值
    • numpy数组运算都会保留索引和值之间的链接
    • 可以将Series堪称是一个定长的有序字典(索引值到数据值的一个映射)
    • 也可以通过Python字典来创建Series(一个字典：索引就是键)
    • isnull和notnull函数可用于检测缺失数据
    • 它在算数运算中会自动对齐不同索引的数据
    • Series对象本身及其索引都有一个name属性,该属性跟pandas其他的关键功能关系非常密切
    • Series的索引可以通过赋值的方式就地修改

  • DataFrame

    • 一个表格型的数据结构,既有行索引也有列索引
    • 数据是以一个或多个二维块存放的(而不是列表、字典或别的一维数据结构)
    • 构建DataFrame
      • 直接传入一个由等长列表或numpy数组组成的字典
      • 如果指定了列序列(colums=””),列就会按照指定顺序进行排列
      • 传入的列在数据中找不到就会产生NA值
    • 通过类似字典标记的方式或属性的方式获取DF列为一个Series
    • 返回的Series拥有原DF相同的索引且name属性也被设置好了
    • 列可以通过赋值的方式进行修改
    • 将列表或数组赋值给某列时,其长度必须跟DF长度相匹配
    • 为不存在的列赋值会创建出一个新列
    • 关键字del用于删除列
    • PS：通过索引方式返回的列只是相应数据的试图而已,并不是副本
    • 参数传入嵌套字典(外层字典的键作为列,内层键则作为行索引)
    • DataFrame构造函数所能接受的各种数据(表5-1)
    • 可以设置index和columns的name属性,会显示出来
    • df.values属性以二维ndarray形式返回数据
    • df各列数据类型不同,则数组数据类型就会选择能兼容所有列的数据类型

  • 索引对象

    • 负责管理轴标签和其他元数据(轴名称etc)
    • 构建Series和DF,所用到的任何数组或其他的序列的标签都会被转换成一个Index
    • Index对象是不可修改的
